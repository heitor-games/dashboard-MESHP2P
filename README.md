<h1>How to Use/Como usar</h1>

1. Clone o repositório caso não tenha os arquivos de projeto no servidor.
2. Dê um `npm install` na raiz do projeto via terminal da sua IDE ou via prompt de comando.
  - Para rodar o projeto em modo de desenvolvimento digite `npm run dev` no terminal.
  - Para buildar o projeto base do servidor digite `npm run build` no terminal.
  - Para gerar um Apk com as ultimas modificações utilize `npm run genApk` no seu terminal.
3. Pronto! Seu projeto está rodando!
